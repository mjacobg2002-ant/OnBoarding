# Daily Time Blocks + Scorecard

---

## Daily Time-Block Template

Adjust dial-block length as weekly dial targets ramp (see `14_30Day_Ramp_Plan.md`), but keep the shape consistent — consistency is what makes 100+ dials/day sustainable for a remote rep with no one watching over their shoulder.

| Time | Block |
|---|---|
| 8:30–9:00 | Daily Startup — review HubSpot tasks/follow-ups due today, check calendar for appointments, 5-min solo roleplay warm-up |
| 9:00–11:00 | **Prime Dial Block 1** |
| 11:00–11:30 | Lead Sourcing Top-Up — replenish prospect list via Google Maps/search/directories, log into HubSpot |
| 11:30–12:30 | Scheduled Appointments (Discovery or Close) — if none scheduled, continue dialing |
| 12:30–1:15 | Lunch |
| 1:15–3:15 | **Prime Dial Block 2** |
| 3:15–3:45 | Follow-Up Touches — SMS/email/LinkedIn per the multi-touch cadence, log every touch |
| 3:45–4:30 | Afternoon Appointments — if none scheduled, continue dialing |
| 4:30–5:00 | Wrap-Up — HubSpot hygiene (log all calls/notes), submit any Marcel Handoff Summaries outstanding, fill Daily Scorecard, plan tomorrow's top prospects |

**Non-negotiable:** Prime Dial Blocks are protected. Admin, sourcing, and follow-up touches happen around them, not instead of them.
