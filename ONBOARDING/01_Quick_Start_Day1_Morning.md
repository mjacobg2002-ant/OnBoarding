# Quick Start — Day 1 Morning
### One page. What actually happens first.

This assumes the **3-Day (Recommended)** track. If running the 2-Day track instead, the morning looks the same — see `13_Onboarding_Agenda_2Day.md` for what shifts in the afternoon.

---

## Before Day 1 (Marcel/Trainer — do this the day before)

- [ ] HubSpot seat provisioned for the new hire (see `16_HubSpot_Pipeline_and_Lead_Tracking.md` for the properties/pipeline they'll need access to)
- [ ] Phone/dialer access set up (whatever the current calling tool is)
- [ ] Stripe payment link / contract template access confirmed
- [ ] This entire folder shared with the new hire
- [ ] Calendar hold sent for Day 1–3 training blocks
- [ ] Trainer (Marcel or delegate) has read `21_Manager_Kit_Facilitator_Guide.md`

---

## Day 1 Morning — Hour by Hour

| Time | Block | What Happens |
|---|---|---|
| 8:30–8:45 | Welcome + Logistics | Access check: HubSpot, dialer, this folder. Confirm comms channel (Slack/text/email) for questions during training. |
| 8:45–9:30 | Company + Offer | Read and discuss `02_Company_Offer_Reference.md` together. New hire should be able to say the three website tiers, three retainer tiers, and the process flow from memory by 9:30. |
| 9:30–10:15 | Who We Sell To | Read `03_ICP_and_Qualifying_Framework.md`. Introduce the **LEAK Score**. New hire should be able to explain all four letters unprompted. |
| 10:15–10:30 | Break | — |
| 10:30–11:30 | Cold Calling Fundamentals | Read `04_Cold_Calling_Fundamentals.md`. Walk the 6-Step Cold Call Flow out loud. Trainer models one full call live (any niche). |
| 11:30–12:30 | First Reps | New hire runs 3–5 **Basic** roleplays from `17_Roleplay_Bank_All_Levels.md` with the trainer playing the prospect. Score using `18_Evaluation_Rubrics_and_Certification.md` (cold call rubric only, today). |

---

## What "Done" Looks Like By Lunch on Day 1

- [ ] New hire can recite the offer stack and process flow without notes
- [ ] New hire can explain LEAK and score a mock prospect
- [ ] New hire has run at least 3 Basic roleplays and scored 3+/5 average on the cold call rubric
- [ ] New hire knows where every file in this folder lives and what it's for

If any box isn't checked, don't move to the afternoon block — repeat the relevant morning section. Speed-to-productivity only works if the foundation actually holds.
