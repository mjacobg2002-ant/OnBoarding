# 30-Day Ramp Plan

---

## Overview

| Period | Dials/Day | Appointments/Day |
|---|---|---|
| Weeks 1–2 | 60–100 | ~1 |
| Month 1 (Weeks 3–4) | 100–150 | ~2 |
| Month 2+ | 120–200 | ~3 average |

These are the same activity targets from `02_Company_Offer_Reference.md`, broken into a week-by-week plan with milestones and manager check-ins attached.

---

## Week 1

| | |
|---|---|
| **Dial Target** | 60–100/day |
| **Appointment Target** | Building toward ~1/day by end of week |
| **Milestones** | First live dials (Day 4 if 3-Day trained, Day 3 if 2-Day trained). First 5 real Marcel Handoff Summaries submitted and reviewed for quality. First HubSpot pipeline hygiene check. |
| **2-Day Track Add-On** | Extra 30–45 min of roleplay practice each morning before live dialing, through end of Week 1, to compensate for the compressed initial training. |
| **Manager Check-In** | End of Week 1: review Daily Scorecards, listen to 2–3 recorded calls (or shadow live), score against `20_Call_QA_Checklist.md`. |

---

## Week 2

| | |
|---|---|
| **Dial Target** | 60–100/day |
| **Appointment Target** | ~1/day |
| **Milestones** | First Close/Presentation Call run. First contract sent. Rep operating with minimal trainer shadowing. |
| **Manager Check-In** | Mid-week: spot-check 2 Marcel Handoff Summaries for completeness. End of week: full scorecard review + 1:1. |

---

## Weeks 3–4 (Month 1)

| | |
|---|---|
| **Dial Target** | 100–150/day |
| **Appointment Target** | ~2/day |
| **Milestones** | First deal closed (contract + deposit collected). Full independent HubSpot hygiene (no reminders needed). Rep can run both Discovery and Close calls without a script in front of them. |
| **Manager Check-In** | Weekly scorecard review (see `15_Daily_Time_Blocks_and_Scorecard.md` for the rollup format). Recalibrate LEAK-qualifying accuracy — are booked appointments actually LEAK-qualified, or is quantity creeping ahead of quality? See `19_Failure_Modes_and_Corrections.md`. |

---

## Month 2+

| | |
|---|---|
| **Dial Target** | 120–200/day |
| **Appointment Target** | ~3/day average |
| **Milestones** | Full autonomy — sourcing, outreach, discovery, close, all self-managed. Consistent multi-touch cadence discipline (no leads going stale from missed follow-ups). |
| **Manager Check-In** | Shift from daily/weekly hand-holding to weekly scorecard review + monthly deeper QA pass (random call review against `20_Call_QA_Checklist.md`). |

---

## What To Do If a Rep Is Behind Target

1. Pull the Daily Scorecard (`15_Daily_Time_Blocks_and_Scorecard.md`) — is the shortfall in **dial volume** (activity problem) or **appointment conversion** (qualifying/skill problem)?
2. If activity: review time-blocking adherence — is the rep actually protecting dial blocks, or getting pulled into admin/distractions?
3. If conversion: pull 2–3 recent recorded calls and QA them against `20_Call_QA_Checklist.md`. Cross-reference `19_Failure_Modes_and_Corrections.md` for the specific pattern.
4. Run 2–3 targeted roleplay reps on the specific weak skill (opener, qualifying, objection handling, or closing) from `17_Roleplay_Bank_All_Levels.md`.
5. Re-check in 1 week — don't wait a full month to intervene.
