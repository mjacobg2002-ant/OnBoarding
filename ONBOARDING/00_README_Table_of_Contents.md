# Kind Supply Digital — SDR/Closer Training Package
### Start Here

This package is a complete, execution-ready onboarding + training system for a new full-cycle SDR/Closer at Kind Supply Digital (KSD). It is built for a **remote rep with minimal hand-holding** — every file is designed to be usable on its own, without someone standing over the rep's shoulder.

---

## Setup Assumptions Used to Build This Package

| Question | Answer Used |
|---|---|
| Which rollout leads first? | **3-Day (Recommended)** track leads. The 2-Day (Compressed) track is fully built too, as a standing alternative for faster starts or repeat hires who need speed over depth. |
| Lead tracking system | **HubSpot** — all pipeline, property, and reporting guidance is written for HubSpot, not a generic Google Sheet. |
| Paid ads in the offer? | **Yes** — paid ads (Google/Meta) are treated as part of the Growth and Premium Growth retainer conversation. Exact ad-spend/management packaging is flagged as an open item for Marcel (see punch list below). |

If any of these change, the files most affected are: `12_Onboarding_Agenda_3Day.md` / `13_Onboarding_Agenda_2Day.md` (rollout choice), `16_HubSpot_Pipeline_and_Lead_Tracking.md` (CRM), and `02_Company_Offer_Reference.md` + all three niche playbooks (paid ads).

---

## How To Use This Package

**If you're the new hire:** Start with `01_Quick_Start_Day1_Morning.md`, then follow `12_Onboarding_Agenda_3Day.md` hour by hour. Everything it references lives in this folder — you should never need to guess.

**If you're Marcel (or training a future hire):** Start with `21_Manager_Kit_Facilitator_Guide.md`. It tells you how to run this whole program, how to grade, and what "certified to go live" actually means.

## Open Items — "Confirm with Marcel" Punch List

Every file is fully usable as-is, but a few specifics weren't defined in the source brief and are flagged inline wherever they show up. Consolidated here so nothing gets missed before go-live:

- [ ] Exact commission structure (rates/tiers) for the commission-only, uncapped comp plan
- [ ] Exact deposit amount/percentage and Stripe invoicing steps for contracts
- [ ] Exact inclusions per website tier (Starter $3,000 / Standard $8,000 / Premium $20,000+) beyond the directional scope given in `02_Company_Offer_Reference.md`
- [ ] Which retainer tier(s) include managed paid ad spend, which ad platforms (Google Ads/LSA/Meta), and how ad budget is billed (client-paid vs. agency-managed)
- [ ] Who owns fulfillment/delivery after a deal closes (confirms the "what happens after Closed Won" step)
- [ ] Whether your current HubSpot portal already has pipeline stages/custom properties that differ from the recommended setup in `16_HubSpot_Pipeline_and_Lead_Tracking.md`
- [ ] Sign-off on the SMS/email compliance approach in `11_Outreach_Templates_Multichannel.md` (TCPA/CAN-SPAM posture)
